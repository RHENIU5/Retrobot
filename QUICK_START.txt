## ⚡ Quick Commands

### Start Server
```bash
cd /Users/rheniusj/Projects/Retrobot/node_chatbot
npm start
```

### Test Chat API
```bash
curl -X POST http://localhost:3000/api/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Write a Python hello world"}'
```

### Stop Server
```bash
killall node
```

### View Logs
```bash
cat /tmp/server.log
```

## 🔧 Configuration

Edit `.env` to change:
```
PORT=3000                          # Server port
OLLAMA_ENDPOINT=http://localhost:11434
OLLAMA_MODEL=deepseek-llm          # Chat model
IMAGE_MODEL=stable-diffusion       # Image model
```

## 🎓 How Mode Detection Works

```
User Input
    ↓
[Check patterns/keywords]
    ↓
Detect Mode (coding, notes, etc.)
    ↓
Enhance prompt with mode-specific instructions
    ↓
Send to Ollama
    ↓
Return response with mode label
```

## 💡 Tips & Tricks

1. **Better Responses**: Be specific with your requests
2. **Code Mode**: Say "write code for..." or "debug this..."
3. **Study Help**: Say "help me solve..." or "explain this..."
4. **Quick Summary**: Say "summarize in bullet points..."
5. **Image Creation**: Start with "generate image of..."

## 📞 Troubleshooting

| Problem | Fix |
|---------|-----|
| Port 3000 in use | `killall node` or change PORT in .env |
| Ollama offline | Run `ollama serve` in another terminal |
| Model not found | `ollama pull deepseek-llm` |
| No image generation | `ollama pull stable-diffusion` |
